# Sound Files

This directory should contain the following sound files for the interval timer:

## Required Files

1. **ready.wav** - Setup countdown sound (~3 seconds max)
2. **start.wav** - Active interval start sound (~0.5-1 second)
3. **complete.wav** - Active interval end sound (~0.5-1 second)
4. **school-bell.mp3** - Rest start sound (used when entering rest period)
5. **cooldown.ogg** - Cooldown interval sound (~0.5-1 second)

The app runs without these files; sounds will simply not play until you add them.

Free sources: freesound.org, zapsplat.com, pixabay.com.
