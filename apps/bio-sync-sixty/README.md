# Bio-Sync Sixty

Astro app for the Bio-Sync Sixty (90-90-1) experience in the interval-timers monorepo.

## Run locally

**Prerequisites:** Node.js

From this app directory:

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev` (serves at port 5177)

From the monorepo root:

- `npm run dev:bio-sync-sixty` — start dev server
- `npm run build:bio-sync-sixty` — production build

See [docs/COMMANDS.md](../../docs/COMMANDS.md) for full workspace commands.
