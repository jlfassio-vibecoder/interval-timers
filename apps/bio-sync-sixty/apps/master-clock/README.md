# Master Clock

Align your metabolic window with seasonal daylight for the Bio-Sync60 Metabolic Switch.

This app is built and deployed as part of the Sync60 project at **/bio-sync60/master-clock** (see repo root `build:deploy` and `scripts/copy-standalone-apps-to-dist.cjs`).

## Run Locally

From the repo root:

- `npm run dev -w master-clock` (or `npm run dev:master-clock`)

From this directory:

1. `npm install`
2. `npm run dev`
