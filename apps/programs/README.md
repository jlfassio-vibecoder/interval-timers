# AI Fit Copilot

The AI-Powered Operating System for Modern Trainers.

This repository is a fork of `ai-fitness-guy`, repurposed as a multi-user coaching platform.

## Features

- **Trainer View:** Program design, client management, analytics.
- **Client View:** Workout execution, progress tracking.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set required env vars in `.env.local` (Firebase, reCAPTCHA, Gemini, Runway; see PRE_PR_CHECKLIST.md). Run `npm run check-env` to validate.
3. Run the app:
   `npm run dev`
