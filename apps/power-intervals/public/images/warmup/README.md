# Warm-up exercise images

Add image files here for the Daily Warm-Up timer. They are served at `/images/warmup/<filename>` and shown in the **top right of the timer header** for each exercise.

See **MISSING_IMAGES.md** for the list of warmup image filenames required by the app. Images are selected by matching exercise names against the patterns defined in `WARMUP_IMAGE_MAP` in `packages/timer-core/src/interval-timer-warmup.ts`, so multiple exercises may share the same image; add the files with the exact names listed there.
